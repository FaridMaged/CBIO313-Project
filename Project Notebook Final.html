Section One - Initial Cleaning [Dropping Empty Columns, Columns with no unique values, low variance columns, and correctly encoding some of the values in the drug columns. ]

Dropping Irrelevant Columns: Targeted columns with a huge numbe of missing values or irrelevant to the study (e.g. encounter ID is just an identifier that would not contibute, kept patient identifier as as the same patient could come in for a vist multiple times, keeping it would be essential for contributing to the model)

In[22]:
df.drop(columns=['encounter_id', 'weight', 'payer_code', 'medical_specialty'], inplace=True)

After inspecting the paper which this dataset had come from, I decided to drop the columns with either a very high number of missing values, or no unique values (meaning that it would no contribute to the machine learning model at all). Note that I did not drop the patient number column, as the same patient could come in for multiple encounters. Keeping this data is valuable to keep the model unbiased.

It seems that the Max Glucose Serum and the A1c test results columns have 'None' values that are recognized as null. However, after inspecting the paper, these values indicate that the test has not been performed on the respective patient, and does not correlate to null values. To prevent pandas from recognizing these values as null, I will encode the 'None' into 'NoTest'

In[25]:
df['max_glu_serum'] = df['max_glu_serum'].replace('None', 'NoTest')

For the age feature, I decided that the best option would be to convert the age ranges to its respective midpoint integer, on the csv file they are written as ranges (e.g. [0-10)), so encoding it as 5 would make it easier to deal with.

from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.impute import SimpleImputer
from sklearn.preprocessing import OneHotEncoder, StandardScaler

#identify column types
numeric_features = X_train.select_dtypes(include=['int64', 'float64']).columns.tolist()
categorical_features = X_train.select_dtypes(include=['object']).columns.tolist()

#numeric pipeline
numeric_transformer = Pipeline(steps=[
    ('imputer', SimpleImputer(strategy='median')),
    ('scaler', StandardScaler())
])

#categorical pipeline
categorical_transformer = Pipeline(steps=[
    ('imputer', SimpleImputer(strategy='constant', fill_value='missing')),
    ('onehot', OneHotEncoder(handle_unknown='ignore'))
])

#combined preprocessor
preprocessor = ColumnTransformer(transformers=[
    ('num', numeric_transformer, numeric_features),
    ('cat', categorical_transformer, categorical_features)
])

Phase two and three

First Model to test: Random Forest Hyperparamaters to tune:

  • n_estimators: Number of trees.
  • max_depth: Max depth of trees.
  • min_samples_split: Minimum samples required to split.

To efficiently select the best hyperparameters, the following steps will be taken:

  • Training it with different hyperparameters using a for loop.
  • Evaluating performance on the test set using metrics like accuracy, precision, recall, and F1-score.
  • Plotting a scatter plot of each hyperparameter combination with a selected metric.
In[37]:
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score
import matplotlib.pyplot as plt
import itertools
import pandas as pd

# Define hyperparameter grid
n_estimators = [50, 100, 150]
max_depth = [5, 10, None]
min_samples_split = [2, 5]

# Create a list of all combinations
param_combinations = list(itertools.product(n_estimators, max_depth, min_samples_split))

# Store results
results = []

for n, d, s in param_combinations:
    clf = RandomForestClassifier(n_estimators=n, max_depth=d, min_samples_split=s, random_state=42)
    clf.fit(X_train_processed, y_train)

    y_pred = clf.predict(X_test_processed)

    acc = accuracy_score(y_test, y_pred)
    prec = precision_score(y_test, y_pred, average='weighted', zero_division=0)
    rec = recall_score(y_test, y_pred, average='weighted', zero_division=0)
    f1 = f1_score(y_test, y_pred, average='weighted', zero_division=0)

    results.append({
        'n_estimators': n,
        'max_depth': d if d is not None else "None",
        'min_samples_split': s,
        'accuracy': acc,
        'precision': prec,
        'recall': rec,
        'f1_score': f1
    })

# Convert to DataFrame
results_df = pd.DataFrame(results)

# Sort results by recall descending
results_df_sorted = results_df.sort_values(by='recall', ascending=False)
print(results_df_sorted.head())

from xgboost import XGBClassifier
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score
import itertools
import pandas as pd

# Define hyperparameter grid
n_estimators = [50, 100, 150]
max_depth = [3, 6, 10]
learning_rate = [0.01, 0.1, 0.2]

# Create list of all combinations
param_combinations = list(itertools.product(n_estimators, max_depth, learning_rate))

# Store results
xgb_results = []

for n, d, lr in param_combinations:
    clf = XGBClassifier(
        n_estimators=n,
        max_depth=d,
        learning_rate=lr,
        random_state=42,
        verbosity=0  # disables training logs
    )
    
    clf.fit(X_train_processed, y_train)

        'n_estimators': n,
